## 📄 Resume Optimization Notes
- **Length:** Fresher → 1 page, Experienced → 2 pages  
- **Format:** Clean, ATS-friendly, no fancy design/icons  
- **Style:** Active voice, action verbs (*Developed, Automated, Built…*)  
- **Impact:** Results with numbers (e.g., “Reduced page load time by 40%”)  
- **Projects:** Clear explanation of tech + role + results  
- **Avoid:** Profile photo, unnecessary certificates, irrelevant skills, grammar mistakes  

---

## 📈 LinkedIn Optimization Notes
- Professional profile photo + catchy headline  
- About section → concise intro: *what you do, what you’re learning, what role you want*  
- Projects & internships लिखो resume-style format में  
- Regular posts → projects, weekly learnings, GitHub updates  
- Network building → HRs, founders, developers से connect + engage  
- **Golden Rule:** सिर्फ scroll मत करो → engage करो और value add करो

## 🔥 Quick Recap Table

| Area          | Do This ✅                      | Don’t Do This ❌          |
|---------------|--------------------------------|---------------------------|
| Resume Length | 1–2 pages                      | 3+ pages                  |
| Projects      | Role + Tech + Result           | सिर्फ topic लिखना         |
| Voice         | Active                         | Passive                   |
